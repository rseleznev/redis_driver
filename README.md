# Redis driver

Учебный драйвер с минимальным функционалом, реализован через прямые системные вызовы и epoll. Поддерживается только Linux.
Можно использовать конкурентно, но все потоки проходят последовательно через одну точку синхронизации.
Использует RESP3 по умолчанию

Для обработки RESP-формата использует промежуточную модель данных

Доступны команды:

* Ping - проверка соединения
* Hello3 - проверка соединения, включает RESP3 и возвращает некоторую информацию о клиенте
* SetValueForKey - устанавливает значение для ключа, поддерживаются только строки или массивы байт
* GetValueByKey - возвращает значение по ключу, возвращает строку или массив байт
